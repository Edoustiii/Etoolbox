## 📝 یادداشت‌های اضافی

اگر مشکلی مشاهده کردید:

1. کش مرورگر را پاک کنید: `Ctrl + Shift + R`
2. سرور را ری‌استارت کنید: `Ctrl + C` سپس `npm run dev`
3. `node_modules` را پاک و دوباره نصب کنید:
   ```powershell
   Remove-Item -Recurse -Force node_modules
   npm install
   ```
